## 🚀 Projects

### 🐍 [ReconLisk](https://github.com/FMNowacki/ReconLisk)
An NMAP-inspired Python network reconnaissance tool — no root privileges required.
- TCP/UDP scanning with service detection for 6+ services
- Configurable scan profiles with full user customisation and automation
- *Planned:* architecture redesign and SYN scanning support

### 🐉 [Hydralert](https://github.com/FMNowacki/Hydralert)
A Flask-based fraud detection suite with an interactive web dashboard.
- Supports Random Forest and Gradient Boosting with single-model or combined inference
- Nuanced synthetic transaction generation (high-value, phishing-style, slow-drain patterns)
- Single-transaction scoring, account timeline analysis, and rich exports (CSV/JSON/TXT)
- Full train/predict workflows end-to-end with a model management UI
